# React Hooks Practice

This project contains hands-on examples of all important React Hooks.

Built using **React + Vite**.

## Hooks Covered

- useState
- useEffect
- useRef
- useMemo
- useCallback
- useContext
- useReducer
- Custom Hooks

## Project Structure

Each hook has its own folder inside `src/examples`.

Example:

src/examples/useStateExample
src/examples/useEffectExample
src/examples/useRefExample

## Run Project

npm install
npm run dev

## Purpose

This project was built to practice React hooks and prepare for frontend interviews.
